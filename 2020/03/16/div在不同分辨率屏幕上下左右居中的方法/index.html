如何使一个div在不同分辨率屏幕上下左右居中

发表于 2020-03-16 | 分类于 CSS

方法一：

盒子的margin减去1/2的宽高才能使盒子的正中间居中，不然居中的只是盒子的左上角。

div{
        position:absolute;
        width: 200px;
        height:200px;
        background:red;
        top:50%;
        left:50%;
        margin-left:-100px;
        margin-top:-100px;
    }

方法二：

div{
               position:absolute;
            width: 200px;
            height:200px;
            background:red;
            top: 0;
            left: 0;
            bottom: 0;
            right: 0;
             margin:auto;
    }
