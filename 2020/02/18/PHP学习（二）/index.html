PHP关联数组的遍历方法

php关联数组的遍历,使用foreach方法
代码：
$array2 = array(‘name’=>’mhs’,’age’=>18,’gender’=>’male’);

foreach($arry as $key => $value){
    echo $key.'----'.$value;
}

输出

name----mhsage----18gender----male

二维数组

数组内包含数组即可

$starArry = array(
    array('name'=>'abc','age'=>'15','gender'=>'male'),
    array('name'=>'qwe','age'=>'18','gender'=>'female'),
    array('name'=>'asd','age'=>'7','gender'=>'male'),
    array('name'=>'zxc','age'=>'22','gender'=>'female')
);

for循环遍历数组：

for ($i=0;$i<count($starArry);$i++){
    echo '<span>'.$starArry[$i]['name'].'</span>--'.$starArry[$i]['age'].'---'.$starArry[$i]['gender'].'<br>';
    }
//输出结果
abc--15---male
qwe--18---female
asd--7---male
zxc--22---female

PHP中输出html代码

需要注意的是如果在php文件中直接编写html代码，编译器是不会解析html代码的，输出到浏览器中时会直接包含代码极其样式，需要注意的是，html代码必须写在php代码标签的外面

例：

<style></style> // css代码 
<?php

?>
<span><a>ima</a></span>

在php文件中引入php文件

方法：
在标签中添加php标签，然后再php标签内include’filePath/fileName.php’即可。

<html>
<body>
    <?php
        include 'path/fileName.php';
    ?>
</body>
</html>
