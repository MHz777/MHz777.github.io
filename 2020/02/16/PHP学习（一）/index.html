PHP学习（一）

发表于 2020-02-16 | 分类于 PHP学习

一、 什么是PHP？

最近因为要做毕设所以在自学PHP，这篇文章记录自己的学习过程。

PHP即“超文本预处理器”，是一种通用开源脚本语言。PHP是在服务器端执行的脚本语言，与C语言类似，是常用的网站编程语言。PHP独特的语法混合了C、Java、Perl以及 PHP 自创的语法。利于学习，使用广泛，主要适用于Web开发领域。

二、基础语法

  • 设置编码格式

    header('content-type:text/html;charset=utf-8');

  • 输出方法 echo

    类似于JavaScript中的document.write() 

    echo"hello world!";//echo中使用.拼接字符串
例:
echo 'abc'.'bcd'; //abcbcd

    注：php文件命名不可使用中文，不然会报错。

  • 定义变量

    定义方式：$ + ValueName;

    $num = 1;

  • if语句

语法规则和其他语言相同

$male = false;
if ($male == false){
    echo "人家是女孩纸";
}else{
    echo "哥们是纯爷们";
}
  • switch语句

规则相同

$day = "monday";
switch($day){
    case 'monday':
    echo '上班';
    break;
    case 'Tuesday':
    echo '上班';
    break;
    case 'Wednesday':
    echo '上班';
    break;
    case 'Thursday':
    echo '上班';
    break;
    case 'Friday':
    echo '上班';
    break;
    case 'Saturday':
    echo '依旧上班';
    break;
    case 'Sunday':
    echo '终于可以休息了';
    break; 
}
  • for循环

在php中for循环除了定义变量方法不同其他都相同，while循环和dowhile循环语法相同不再阐述

for($i=1;$i<10;$i++){
    echo '<br>';
    echo '感觉自己萌萌哒'.$i;

}

  • 数组

    定义方式:
    $arryName = array(element);

    $array1 = array('1','asdasd',2);

注:使用echo不能遍历数组
遍历数组使用print_r()方法,或者for循环,在php中数组没有arr.length方法，使用count(arrName)方法获取数组长度

print_r($array1);//Array ( [0] => 1 [1] => asdasd [2] => 2 )

echo $array2[name];//mhs

//遍历普通数组
for($i=1;$i<count($array1);$i++){
echo '<br>';
echo $array1[$i];
}
  • PHP 关联数组

关联数组是使用您分配给数组的指定键的数组。有两种创建关联数组的方法：

$array2 = array('name'=>'mhs','age'=>18,'gender'=>'male');

或者

$array2['name']="mhs";
$array2['age']="18";
$array2['gender']="male";
