PHP中提交数据

在from表单中submit提交数据有两种方式：get 和 post

今天只学习get方式，此时有一个html页面，其中有一个表单

<form action="./getData.php" method="GET" >
    <input name="lol_hero" type="text" placeholder="请输入要查询的英雄">
    <input type="submit" value="查询">
</form>

此时点击查询，则会将文本框中输入的信息提交到”./getData.php”的页面，那么，怎么来接收数据呢？

在PHP中提供了超全局变量 $_GET，来接收get方法所提交的数据，$_GET和js中的window对象类似 ,不需要定义可直接使用
例：

print_r($_GET);
//champion_name=>凯隐

如何从$_GET中取出数据呢

和关联数组取出数据的方法一样，只需要 $_GET[‘TagName’]即可取出get方法所提交的数据。

注: TagName是指input标签中name属性的值

二维数组

数组内包含数组即可

$starArry = array(
    array('name'=>'abc','age'=>'15','gender'=>'male'),
    array('name'=>'qwe','age'=>'18','gender'=>'female'),

    array('name'=>'zxc','age'=>'22','gender'=>'female')

for循环遍历数组：

for ($i=0;$i<count($starArry);$i++){
    echo '<span>'.$starArry[$i]['name'].'</span>--'.$starArry[$i]['age'].'---'.$starArry[$i]['gender'].'<br>';
    }
//输出结果
abc--15---male
qwe--18---female
asd--7---male
zxc--22---female

PHP中输出html代码

需要注意的是如果在php文件中直接编写html代码，编译器是不会解析html代码的，输出到浏览器中时会直接包含代码极其样式，需要注意的是，html代码必须写在php代码标签的外面

例：

<style></style> // css代码 
<?php

?>
<span><a>ima</a></span>

在php文件中引入php文件

方法：
在标签中添加php标签，然后再php标签内include’filePath/fileName.php’即可。

<html>
<body>
    <?php
        include 'path/fileName.php';
    ?>
</body>
</html>

一、 什么是PHP？

最近因为要做毕设所以在自学PHP，这篇文章记录自己的学习过程。

PHP即“超文本预处理器”，是一种通用开源脚本语言。PHP是在服务器端执行的脚本语言，与C语言类似，是常用的网站编程语言。PHP独特的语法混合了C、Java、Perl以及 PHP 自创的语法。利于学习，使用广泛，主要适用于Web开发领域。

二、基础语法

  • 设置编码格式

    header('content-type:text/html;charset=utf-8');

  • 输出方法 echo

    类似于JavaScript中的document.write() 

    echo"hello world!";//echo中使用.拼接字符串
例:
echo 'abc'.'bcd'; //abcbcd

    注：php文件命名不可使用中文，不然会报错。

  • 定义变量

    定义方式：$ + ValueName;

    $num = 1;

  • if语句

语法规则和其他语言相同

$male = false;
if ($male == false){
    echo "人家是女孩纸";
}else{
    echo "哥们是纯爷们";
}
  • switch语句

规则相同

$day = "monday";
switch($day){

  • for循环

在php中for循环除了定义变量方法不同其他都相同，while循环和dowhile循环语法相同不再阐述

    定义方式:

注:使用echo不能遍历数组

echo $array2[name];//mhs

  • PHP 关联数组

关联数组是使用您分配给数组的指定键的数组。有两种创建关联数组的方法：

哈喽你好，这是我的第一篇博客，用来记录markdown的学习过程

一、编辑器的选择

  如markdown、sublime、vscode等等都可以按照个人喜好来选择，我比较懒电脑上有现成的vscode，所以直接使用的是vscode，vscode的优点在此我就不阐述了，感兴趣的话可以去了解一下。


二、行内语法规则

2.1 注释

markdown中支持html标签的方式，所以注释可使用常用的ctrl+/ 的方式，例如：

<!-- 你好我是注释 -->

这一行文字就不会显示在页面上


2.2 分级标题、任务列表

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题  <!--最多6级标题-->

示例效果如下：

   一级标题

    二级标题

    三级标题

    四级标题

    五级标题
    六级标题

2.3 任务列表

markdown语法：

- 未做任务 `未做任务`
- 已做任务 `已做任务`

效果

  • 未做任务 未做任务
    • 已做任务 已做任务

说实话我也不明白这个’’到底是干什么用的，但是效果就是这样，嘻嘻

2.4 文字粗体斜体删除线

*斜体*
**粗体**
***粗斜体***
~~删除线~~

  斜体
     粗体
    粗体斜体     删除线

2.5 缩进、换行、对齐方式

【1】 &emsp;或&#8195; //全角
【2】 &ensp;或&#8194; //半角
【3】 &nbsp;或&#160;  //半角之半角
  • 换行

由于markdown编辑器的不同,可能在一行字后面，直接换行回车，也能实现换行，但是在Visual Studio Code上，想要换行必须得在一行字后面空两个格子才行。

  • 对齐方式

    <center>行中心对齐</center>
<p align="left">行左对齐</p>
<p align="right">行右对齐</p>

2.6 超链接、页内链接、自动链接、注脚

  • 超链接
[linkname](href "title")
[链接名]+(链接)+空格+"title（即鼠标放在链接要显示的字符）"

效果：

请打开 哔哩哔哩动画

或者

<www.bilibili.com> 自动链接

  • 超链接另一种方式:

    [baidu][baidulink]
//中间需要换行
[baidulink]: www.baidu.com

效果:

baidu

  • 注脚

语法说明：

在需要添加注脚的文字后加上脚注名字[^注脚名字],称为加注。 然后在文本的任意位置(一般在最后)添加脚注，脚注前必须有对应的脚注名字。

注意：经测试注脚与注脚之间必须空一行，不然会失效。成功后会发现，即使你没有把注脚写在文末，经Markdown转换后，也会自动归类到文章的最后。

代码：

使用 Markdown[^1]可以效率的书写文档, 直接转换成 HTML[^2]。

[^1]:Markdown是一种纯文本标记语言

[^2]:HyperText Markup Language 超文本标记语言

效果：

使用 Markdown^1可以效率的书写文档, 直接转换成 HTML[^2]。

[^2]:HyperText Markup Language 超文本标记语言
